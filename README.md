## 关于家盈记

家盈记是一款专为家庭设计的资产管理工具：

- 月末录入各账户余额，App 自动推算收支变化
- 12 个月资产走势图，财富增长轨迹一目了然
- 支持多账户、多成员、多币种
- 数据完全本地存储，不联网、不上传
